# 이슈 보고서: SOC-ERR-001 SOC 추정 불일치 알람

## 기본 정보
- **이슈 ID**: BATTERY-2024-009
- **알람 코드**: SOC-ERR-001
- **발생일시**: 2024-10-10 09:05:17
- **심각도**: Medium
- **테스트 단계**: REST (휴지)
- **채널/유닛**: CH-10, CH-12

## 증상
- 충방전 사이클 후 REST 단계 진입 시 SOC-ERR-001 알람 발생
- BMS 표시 SOC: 42% vs 실제 OCV 기반 SOC: 31% (11% 오차)
- 쿨롱 카운팅 누적 오차로 추정
- CH-10, CH-12 동시 발생으로 시스템 공통 원인 의심

## 원인 분석
- BMS 쿨롱 카운팅 전류 센서 오프셋 드리프트 (+28mA 누적)
- 0.028A × 3,600초 = 100.8mAh/cycle 오차 누적
- 약 50사이클 후 SOC 오차가 경보 임계값(10%) 초과
- 전류 센서 온도 보상 계수 미갱신 (초기 설정값 그대로 사용)

## 재현 단계
1. 쿨롱 카운팅 전류 센서 오프셋 측정 (REST 상태에서 순 전류 = 0A 확인)
2. 50사이클 전후 SOC 오차 추이 기록
3. OCV-SOC 룩업 테이블 기반 실측 SOC와 BMS SOC 비교
4. 온도 변화 구간에서 전류 센서 출력 안정성 확인

## 조치 방법
1. 전류 센서 오프셋 보정 (영점 조정 수행)
2. BMS SOC를 OCV 기반 실측값으로 강제 초기화
3. 온도 보상 계수 재설정 (현재 온도 환경 반영)
4. SOC 보정 주기를 50사이클 → 20사이클로 단축

## 테스트 결과
- 보정 후 SOC 오차 ±1.5%로 감소
- 100사이클 후 재측정: ±2.3% 유지

## 영향 범위
- CH-10, CH-12의 SOC 데이터 신뢰도 저하 기간(최대 50사이클) 데이터 플래그 처리
- 동일 BMS 모델 사용 전 채널 전류 센서 점검
