# 이슈 보고서: OTP-003 NTC 센서 드리프트로 인한 과온도 알람

## 기본 정보
- **이슈 ID**: BATTERY-2025-005
- **알람 코드**: OTP-003
- **발생일시**: 2025-03-08 13:45:00
- **심각도**: High
- **테스트 단계**: CC_CHARGE
- **채널/유닛**: CH-02, CH-05, CH-11 (산발적 발생)

## 증상
- CC 충전 중 OTP-003 알람 간헐적 발생 (오경보 의심)
- 알람 발생 시 BMS 온도: 63~68°C, 외부 기준 온도계: 45~49°C
- BMS와 기준 온도계 간 온도 차이: 15~20°C (정상 허용: ±3°C)
- 열화상 카메라로 셀 표면 온도 확인 시 배터리 이상 없음

## 원인 分析
- NTC 서미스터 장기 사용(2년 이상)에 따른 저항값 드리프트
- NTC 특성 곡선 초기 대비 변화로 동일 온도에서 저항값 변화
- 25°C 기준 NTC 저항: 초기 10kΩ → 현재 12.3kΩ (+23% 드리프트)
- R-T 변환 테이블 미갱신으로 오차 누적

## 재현 단계
1. 채널별 NTC 센서 저항값 정밀 측정 (기준 온도 25°C, 항온조 사용)
2. BMS 온도 vs 기준 온도계 대비 오프셋 측정
3. NTC 교체 전후 온도 측정 정확도 비교
4. 사용 기간별 NTC 드리프트 이력 분석

## 조치 방법
1. 2년 이상 사용 NTC 센서 전수 교체
2. NTC 교체 후 R-T 특성 확인 및 BMS 보정값 업데이트
3. NTC 정기 교체 주기 신설: 2년 또는 500사이클
4. BMS에 NTC 드리프트 자가 진단 기능 추가 요청

## 테스트 결과
- NTC 교체 후 온도 측정 정확도 ±1.2°C 복구
- 교체 3개 채널 모두 OTP 알람 미발생 (24시간 모니터링)

## 영향 범위
- 2년 이상 사용 채널 NTC 전수 교체 (총 8채널)
- OTP 오경보 기간 데이터 재검토 및 플래그 처리
