# 이슈 보고서: SOC-JUMP-001 SOC 급격 변화 알람

## 기본 정보
- **이슈 ID**: BATTERY-2024-015
- **알람 코드**: SOC-JUMP-001
- **발생일시**: 2024-12-18 14:32:09
- **심각도**: High
- **테스트 단계**: DISCHARGE
- **채널/유닛**: CH-11

## 증상
- 방전 중 SOC가 65%에서 38%로 순간 급강하 (SOC-JUMP-001 알람)
- 전압, 전류 측정값은 정상 범위 유지
- 약 2초 후 SOC가 63%로 복귀 (BMS 내부 재계산 추정)
- 이 구간에서 충방전 에너지 계산 오류 발생 가능

## 원인 分析
- BMS ADC(아날로그-디지털 변환기) 전류 측정 채널에서 1회성 이상 펄스 발생
- ADC 출력값 순간 -12.7A (실제 +5.3A) 오기록 → SOC 역산 오류
- EMI(전자기 간섭) 유입으로 ADC 레퍼런스 전압 순간 왜곡
- ADC 출력 이상값 필터링 로직 미비

## 재현 단계
1. BMS 전류 데이터 로그에서 이상 펄스 구간 추출
2. 오실로스코프로 ADC 레퍼런스 전압 안정성 모니터링
3. 근처 고전력 장비 동작 시 EMI 수준 측정
4. ADC 이상값 발생 조건 분석

## 조치 방법
1. BMS 펌웨어에 ADC 이상값 필터 추가 (3σ 이상 벗어난 값 제거, 이전 값으로 대체)
2. ADC 레퍼런스 전압 라인에 노이즈 필터 커패시터 추가 (하드웨어 패치)
3. EMI 차폐 강화 (BMS PCB 실드 추가)
4. SOC 급변 이벤트 발생 시 자동 로그 저장 기능 추가

## 테스트 결과
- 펌웨어 업데이트 후 EMI 시뮬레이션 테스트 100회: SOC 점프 0회
- 하드웨어 패치 후 ADC 노이즈 레벨 65% 감소

## 영향 범위
- CH-11 데이터 이상 발생 구간(2초) 데이터 마킹 처리
- 동일 BMS 모델 전 채널 펌웨어 업데이트 실시

## 관련 이력
- BATTERY-2024-020: 그라운드 루프로 인한 통신 오류와 EMI 환경 연관성
- BMS 펌웨어 v2.4.2에 ADC 필터 기능 포함
