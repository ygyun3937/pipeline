# 이슈 보고서: CC-CV-001 CC/CV 전환 전압 불일치 알람

## 기본 정보
- **이슈 ID**: BATTERY-2025-007
- **알람 코드**: CC-CV-001
- **발생일시**: 2025-03-25 16:08:33
- **심각도**: Medium
- **테스트 단계**: CV_CHARGE
- **채널/유닛**: CH-06, CH-14

## 증상
- CC/CV 전환 시점에 전압 오버슈트 발생: 4.232V (목표 전환 전압: 4.200V)
- 전환 시 32mV 초과로 OVP 경보 임박 상태
- 오버슈트 발생 후 CV 단계 시작 전압이 불안정하게 진동
- 이전 펌웨어(v2.3.x)에서는 전환 전압 ±5mV 이내 유지

## 원인 分析
- BMS 펌웨어 v2.4.0 업데이트 후 CC→CV 전환 PID 제어 파라미터 변경
- v2.4.0에서 PID 적분 게인(Ki)이 기존 대비 1.8배 증가
- 높은 Ki로 인해 전환 시점에 과보정(overshoot) 발생
- 펌웨어 업데이트 검증 시 전환 전압 오버슈트 테스트 항목 누락

## 재현 단계
1. v2.4.0과 v2.3.8의 CC/CV 전환 구간 전압 파형 비교
2. PID 파라미터 Ki 값 확인 (v2.3.8 vs v2.4.0)
3. Ki 값을 조정하여 오버슈트 감소 확인
4. 다양한 충전율(0.5C, 1C, 2C)에서 전환 특성 검증

## 조치 방법
1. BMS 펌웨어 v2.4.1 긴급 패치 (Ki 파라미터 v2.3.8 수준으로 롤백)
2. PID 파라미터 변경 시 전환 전압 오버슈트 검증 항목 테스트 체크리스트 추가
3. 향후 PID 자동 튜닝 기능 도입 검토

## 테스트 결과
- v2.4.1 업데이트 후 전환 전압 오버슈트 ±3mV 이내로 개선
- 0.5C, 1C, 2C 전환 특성 모두 정상 확인

## 영향 범위
- 전체 채널 v2.4.1 업데이트 완료
- v2.4.0 운용 기간(1/10~3/25) 충전 데이터 전환 전압 재검토
