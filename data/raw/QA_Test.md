## 1. 이슈 개요 (Issue Overview)

### 1.1 핵심 정보 요약

| 항목 | 내용 |
|------|------|
| **알람 코드** | OVP-001 (과전압 보호 알람) |
| **알람 메시지** | CC 충전 중 과전압 감지 |
| **심각도** | 🔴 Critical |
| **발생 채널** | CH-02 |
| **테스트 단계** | CC_CHARGE (정전류 충전) |
| **알람 발생 시점** | 충전 경과 4,200초 |
| **측정 전압** | 4.35V |
| **OVP 허용 상한** | 4.300V |
| **초과 편차** | +50mV |

| **이슈 등록일** | 2026-03-08 |

### 1.2 증상 설명

CH-02 채널에서 CC_CHARGE(정전류 충전) 단계 진행 중, 충전 시작 후 **4,200초** 시점에 전압 센서가 **4.35V**를 측정하여 OVP 허용 상한(4.300V)을 **+50mV 초과**함으로써 `OVP-001: 과전압 감지 — CC 충전 중단` 알람이 발생하고 충전이 강제 중단되었습니다.

선행 이슈 BATTERY-2024-001의 이력을 참조 시, **실제 배터리 전압은 정상 범위(4.287V)** 였음에도 불구하고 전압 센서 오프셋 누적(+64mV)으로 인해 OVP 오탐(False Positive)이 발생한 동일 패턴이 확인됩니다.

### 1.3 예상 동작 vs 실제 동작

| 구분 | 내용 |
|------|------|
| **예상 동작** | CC_CHARGE 전 구간에서 전압 센서가 실제 배터리 전압을 **±1mV 이내** 오차로 정확히 측정하며, OVP 임계값(4.300V) 도달 전까지 충전이 정상 지속 |
| **실제 동작** | 경과 4,200초 시점에 센서 측정값 **4.35V** 기록 → OVP-001 알람 발생 → 충전 강제 중단. 실제 배터리 전압이 정상 범위일 경우 센서 오프셋 **+50mV 이상** 누적 오차로 인한 오탐 가능성 높음 |

| 이상 채널 분류 | ±5mV 초과 채널 → 즉시 재캘리브레이션 대상으로 분류 |
| 캘리브레이션 주기 | 마지막 수행일이 **3개월 이내** 인지 이력 로그 대조 확인 |
| 결과 산출물 | 채널별 결과 테이블 및 불합격 채널 목록 |
