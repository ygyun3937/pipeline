# 이슈 보고서: UVP-001 과방전 보호 알람

## 기본 정보
- **이슈 ID**: BATTERY-2024-005
- **알람 코드**: UVP-001
- **발생일시**: 2024-08-20 16:44:21
- **심각도**: High
- **테스트 단계**: DISCHARGE (방전)
- **채널/유닛**: CH-05

## 증상
- 정격 방전율 1C로 방전 중 UVP-001 알람 발생 및 방전 즉시 중단
- 측정 전압: 2.481V (하한 임계값: 2.500V 미만)
- 예상 방전 종료 SOC 대비 약 8% 일찍 컷오프
- 알람 발생 전후 전압 강하 패턴이 비정상적으로 가파름

## 원인 분석
- 셀 내부 리튬 도금(lithium plating)으로 인한 국소 저전압 발생
- 이전 저온(-5°C) 환경 급속 충전 이력으로 음극 리튬 도금 형성
- 방전 후반부 도금 영역의 전압 강하가 전체 셀 전압을 끌어내림
- SOC 추정 알고리즘이 실제 방전 용량 감소를 반영하지 못함

## 재현 단계
1. CH-05 배터리를 완전 충전 후 1C 방전 조건 설정
2. 방전 말기(SOC 10~5%) 전압 강하 기울기 모니터링
3. 다른 정상 셀 대비 전압 프로파일 비교
4. 전기화학 임피던스 분광법(EIS)으로 음극 상태 확인

## 조치 방법
1. 해당 셀 EIS 분석으로 리튬 도금 여부 확인
2. 저온 환경 급속 충전 프로토콜 제한 (0°C 이하 시 0.5C 이하로 제한)
3. SOC 추정 모델 파라미터 재보정 (쿨롱 카운팅 + OCV 주기적 보정)
4. UVP 임계값 모니터링 강화 (2.600V에서 조기 경보 추가)

## 테스트 결과
- 조치 후 동일 조건 방전 5회 정상 완료
- SOC 추정 오차 ±2% 이내로 개선

## 영향 범위
- CH-05 단일 채널, 해당 배터리 셀 교체
- 저온 이력이 있는 배터리 전수 점검 및 EIS 진단 실시

## 관련 이력

- BATTERY-2023-029: 유사한 과방전 패턴, 다른 셀 모델에서 발생
