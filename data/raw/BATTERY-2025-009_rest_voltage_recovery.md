# 이슈 보고서: REST-VR-001 REST 단계 전압 회복 이상 알람

## 기본 정보
- **이슈 ID**: BATTERY-2025-009
- **알람 코드**: REST-VR-001
- **발생일시**: 2025-04-15 14:30:55
- **심각도**: Medium
- **테스트 단계**: REST
- **채널/유닛**: CH-12

## 증상
- 방전 종료 후 REST 단계(1시간) 진행 중 REST-VR-001 알람 발생
- 방전 직후 전압: 3.210V → 1시간 후 OCV: 3.389V (회복량 179mV)
- 동일 조건 정상 셀: 1시간 후 OCV 3.621V (회복량 411mV)
- 2시간 휴지 후에도 OCV가 안정화되지 않는 비정상 패턴

## 원인 分析
- 고율 방전(3C) 후 전극 내 리튬 농도 분포 불균일(농도 분극) 심화
- 내부 저항 증가(노화)로 인해 분극 완화(relaxation) 속도 저하
- 현재 배터리: 520사이클, 내부 저항 78mΩ (초기 25mΩ 대비 3.1배)
- SOC 추정 알고리즘이 불완전한 OCV를 사용하여 SOC 오산출

## 재현 단계
1. 1C, 2C, 3C 방전 후 각각 OCV 회복 프로파일 측정
2. EIS로 분극 저항 성분 분리 측정
3. 충분한 휴지(4시간) 후 완전히 안정된 OCV 확인
4. 정상 배터리(100사이클 이하) 대비 OCV 회복 시간 비교

## 조치 방법
1. 고율(2C 이상) 방전 후 REST 시간을 1시간 → 3시간으로 연장
2. OCV 안정화 판단 기준 추가 (30분간 전압 변화 <1mV)
3. 520사이클 배터리 교체 계획 수립 (내부 저항 기준 초과)
4. SOC 추정 모델에 분극 보정 계수 적용 검토

## 테스트 결과
- REST 3시간 설정 후: OCV 3.609V 안정화 확인
- 4시간: 3.618V (0.3mV 변화, 안정 판정 기준 충족)

## 영향 범위
- CH-12 배터리 교체 계획 수립 (우선순위 높음)
- 3C 이상 방전 프로파일 사용 채널 REST 시간 일괄 재검토
