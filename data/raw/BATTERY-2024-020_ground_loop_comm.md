# 이슈 보고서: COMM-GND-001 CAN 통신 그라운드 루프 알람

## 기본 정보
- **이슈 ID**: BATTERY-2024-020
- **알람 코드**: COMM-GND-001
- **발생일시**: 2025-01-10 10:08:42
- **심각도**: High
- **테스트 단계**: CC_CHARGE
- **채널/유닛**: RACK-1, RACK-2 (CAN 버스 공유)

## 증상
- CAN 통신 오류율 갑자기 증가 (오류율 0.1% → 18%)
- 간헐적 BMS 데이터 수신 불가 및 알람 발생
- RACK-1과 RACK-2를 연결하는 CAN 버스 세그먼트에서만 발생
- 새 전력 케이블 설치 작업 직후부터 증상 발생

## 원인 分析
- 신규 전력 케이블 설치 시 RACK-2 접지를 별도 접지 바(bar)에 연결
- RACK-1과 RACK-2의 접지 전위 차이 발생 (측정: 0.8V)
- CAN 버스 공통 모드 전압 허용 범위(±2.5V) 이내이나 노이즈 마진 감소
- 접지 전위 차이로 CAN 버스에 공통 모드 노이즈 유입

## 재현 단계
1. RACK-1, RACK-2 접지 전위 차이 측정
2. CAN 버스 파형 오실로스코프 측정 (차동 및 공통 모드)
3. 두 래크 접지 연결 후 CAN 통신 오류율 변화 확인
4. 전력 케이블 설치 전후 그라운드 토폴로지 비교

## 조치 방법
1. RACK-1과 RACK-2 접지를 동일 메인 접지 바에 연결 (단일 포인트 접지)
2. CAN 버스 격리형 트랜시버(isolated transceiver) 추가 설치
3. 접지 설계 기준 문서화 및 케이블 설치 체크리스트 추가
4. 그라운드 루프 방지를 위한 접지 연결도 도면 갱신

## 테스트 결과
- 단일 포인트 접지 적용 후 CAN 오류율 0.02%로 복구
- 72시간 연속 모니터링: 통신 안정성 유지

## 영향 범위
- RACK-1, RACK-2 CAN 데이터 이상 기간(1/8~1/10) 데이터 검토
- 전체 래크 접지 연결 상태 점검 및 도면 업데이트
